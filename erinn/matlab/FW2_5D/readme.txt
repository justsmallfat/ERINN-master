Readme file for FW_2_5d Electrical Resistivity Forward modeling package

To run the driver file:

Start matlab. Set the present working directory to FW_2_5d. 
type 'Fw_2_5d_driver' and hit return.

The should run in approximatley 5min on a 2Ghz Pentium 4 machine.

Please note, when changing the model size, i.e. the size of dx and dz,
 the conductivity structure 's' must change size to match the nex dx and dz
